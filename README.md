# PHYS 382 - Bridge Simulator
## Bridge Simulation Code
The code is a godot project named bridge stored in the bridge directory in this repository.

## Sources for Material and Bridge Values

Bridge Behavior and Properties

1. Rayleigh Damping Theory: https://www.simscale.com/knowledge-base/rayleigh-damping-coefficients/

Wood Material Properties

1. Density: https://cedarstripkayak.wordpress.com/lumber-selection/162-2/
2. Dimensions: https://www.dot.state.mn.us/bridge/pdf/insp/USFS-TimberBridgeManual/em7700_8_chapter02.pdf

Steel Material Properties

1. Density and Dimensions: https://stavianmetal.com/en/i-beam-steel-weight-chart-mass-standards/

Road Material Properties

1. Density: https://kg-m3.com/material/concrete
2. Dimensions: https://jackwinsafety.com/road-dimensions/
